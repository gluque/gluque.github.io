* D3
  - D3 is a JavaScript library for manipulating *web documents* based on data
  - When you're going to make a chart with D3, you have to deal with HTML, SVG, CSS, and JavaScript
    - *HTML* is a markup language for describing web **documents**
    - *SVG* (Scalable Vector Graphics) allows for the creation of vector based images using an XML format
    - *CSS* (Cascading Style Sheets) is used for describing the presentation of a document
    - *JavaScript* is the language which brings life to web documents
      - It orchestrates the interactions between HTML, SVG, and CSS
