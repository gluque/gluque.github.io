Rails 5.2.3 on AWS

Prerrequisites Create an EC2 instance (Ubuntu 16.04) Install MySQL or provision a Postgres database on AWS RDS Configure Capistrano Add the following gems to your Gemfile: gem 'capistrano', group: :development gem 'capistrano-bundler', group: :development gem 'capistrano-passenger', group: :development gem 'capistrano-rails', group: :development gem 'capistrano-rvm', group: :development Install the gems and Capistrano: bundle install bundle exec cap install Capistrano create a number of files when installed. The single entry point for Capistrano is Capfile which contains the instructions that Capistrano will execute to perform the deployment....
